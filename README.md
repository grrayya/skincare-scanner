# SkinScan CLI 🧴

A terminal-based skincare analyzer that processes facial images to recommend personalized product routines. 

## Features
* **Interactive Profiling:** asks the users for baseline metrics (skin type, sensitivity).
* **Automated Detection (Mocked):** Simulates computer vision severity scores for acne, dryness, hyperpigmentation, and wrinkles.
* **Logic Engine:** Maps detection scores against a localized JSON database to generate a safe product routine.
* **Prescription Export:** Automatically saves the user's final analysis and action plan to a text file.
